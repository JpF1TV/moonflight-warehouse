# Moonflight - Sistema de Control de Almacén

Sistema web para el control de inventario de almacenes de Moonflight.

## Características

- **Login**: Sistema de autenticación con roles
- **Almacén**: Registro de entrada/salida de piezas
- **Historial**: Seguimiento de todos los movimientos
- **Solicitud de Piezas**: Módulo para solicitar piezas del almacén
- **Administrativo**: Aprobación de solicitudes de salida

## Usuarios Demo

- **Admin**: admin / admin123 (acceso completo)
- **Almacén**: almacen / almacen123 (gestión de almacén)
- **Usuario**: usuario / usuario123 (solicitudes)

## Instalación
